What to include

  • SD Import version and build.
  • macOS version and Mac model.
  • Camera/card brand, filesystem, and reader type.
  • Whether the import was automatic or manually started.
  • What the preview showed and what happened after import.

Export diagnostics.

Diagnostics export is opt-in and redacted. It excludes media files, filenames, and full source/destination paths.

1

Open Diagnostics

In SD Import, choose the Diagnostics section from the sidebar.

2

Export

Choose Copy Diagnostics or Export Diagnostics. Review the file before sharing it.

3

Attach selectively

Share only what is needed. Do not attach private media, full card dumps, or credentials.

Crash reports.

SD Import does not upload crash reports automatically. If the app crashes, macOS may store a local crash report that you can choose to share.

~/Library/Logs/DiagnosticReports/

Use Diagnostics, Reveal Crash Reports to open the folder, or Export Latest Crash Report to save the newest local SD Import report. Review it before attaching it to email or an issue.
