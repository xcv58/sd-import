Built for repeated card work.

SD Import focuses on the checks that matter before files move: source, destinations, counts, space, known originals, and sidecars.

Preview before copying

Scan the card, review what will copy, and inspect destination folders before starting the import.

Known files stay skipped

Reinsert the same card without duplicating originals that SD Import already knows.

Sidecars stay visible

Photo imports skip camera support files. Footage backups can keep sidecars when the project needs them.

Free, signed, and updateable.

The public build is distributed through GitHub Releases as a signed and notarized DMG. Sparkle handles in-app updates from the GitHub Release feed.
