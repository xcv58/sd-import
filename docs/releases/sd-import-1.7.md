# SD Import 1.7

## Changes

- Shows clearer destination previews, including grouped target folders, full paths, and free-space warnings before copying.
- Improves History so scan-only jobs stay hidden, rows sort by the displayed job timestamp, and job titles prefer meaningful shoot names over placeholder card names.
- Reduces false prompts from Recovery and other non-card volumes by checking for supported media before prompting.
- Makes scan/import flows safer with cancellable scans, retry/forget guards during active work, and dedupe-aware space checks.
- Improves Footage Backup sidecar handling with flat card-folder copies, default sidecar inclusion, and sidecar dedupe/forget support.
- Adds clearer setup and accessibility copy, including Shoot name labels, path validation in Settings and onboarding, specific folder button labels, and safer destructive confirmations.
