# SD Import 1.18

- Adds one-click recent choices for shoot names, source folders, and photo/video destinations.
- Shows unavailable recent folders as disabled options, so it is clearer when a card or destination is not mounted.
- Adds a native report viewer for import history with import totals, copied/skipped/failed files, warnings, and access to the original report file.
- Improves report reliability by refreshing file details from native history and disabling report-file actions when the original file is missing.
- Refines recent-choice matching and labels, including case-insensitive shoot-name deduping and shorter parent-path context for repeated folder names.
