# SD Import 1.13

- Makes Footage Backup imports truly flat, so videos and selected sidecars land directly in the dated session folder instead of a `Card Untitled` subfolder.
- Keeps duplicate clip names safe in flat folders by renaming later copies with suffixes such as `C0001-copy-1.MP4`.
- Ignores Sony camera index files `MEDIAPRO.XML` and `DATABASE.BIN` during scanning so they do not clutter imported shoot folders.
