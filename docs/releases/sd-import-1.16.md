# SD Import 1.16

- Hides date rows from the import preview when every file for that date is already known and will be skipped.
- Keeps scan totals and skipped-file counts visible, so known files are still accounted for without looking like new import work.
- Counts only import-selectable files in the date/session editor when a date contains a mix of known and new media.
- Keeps manually excluded session rows visible during customization so they can still be re-enabled.
