# SD Import 1.23

- Replaces the import preview's Preset/Customize/Organization controls with direct Import, Destination Layout, and Dates controls.
- Adds an explicit destination layout choice for one shoot folder, separate photo/video folders, and footage backup while keeping existing import behavior compatible.
- Keeps legacy Classic organization support internally, but removes it from the normal user-facing workflow.
- Fixes a settings-save error that could appear when choosing a recent shoot name.
