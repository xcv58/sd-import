# SD Import 1.20

- Refines the import report viewer with a compact top-right close button and a single Original Report menu for opening or revealing the source Markdown report.
- Improves the report file list so failed files and conflicts appear first, followed by copied files, before skipped or unsupported files.
- Adds Manage Recent Folders sheets for source and destination pickers, with per-folder Use and Forget actions.
- Lets users hide unwanted recent folders from suggestions without deleting import history, and restore forgotten folders later.
