# SD Import 1.9

## Changes

- Refreshes the main app interface with clearer sections, denser controls, and more polished Import, History, Settings, and Diagnostics screens.
- Adds a Theme setting with System, Light, and Dark options; System remains the default.
- Fixes startup Diagnostics errors when a previously selected SD card or source bookmark is no longer mounted.
- Keeps missing cards as normal validation feedback, so users see `Card is not mounted` instead of a raw macOS file error.
