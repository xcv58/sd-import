# SD Import 1.14

- Lets you scan a card with only the source selected, then asks only for the destination needed by the detected photo, video, or mixed workflow.
- Makes Photos and Videos destinations optional during setup so photo-only and video-only workflows can get started faster.
- Removes the confusing top-right Scan and Import toolbar icons from the Import screen in favor of the clearer labeled workflow buttons.
- Simplifies the History toolbar so Refresh stays visible while Retry remains in the selected job details.
- Keeps large History file lists contained in their own scroll area with a visible file count, so thousand-file imports no longer make the whole pane hard to navigate.
