# SD Import 1.8

## Changes

- Adds a Folder grouping option so imports can stay split by day or go into one flat shoot folder across multiple days.
- Uses date-range folder names for multi-day shoot folders, such as `2026-05-02 to 2026-05-04 Launch Weekend`.
- Renames duplicate filenames within a flat shoot import before copying so camera filename resets do not overwrite files.
- Adds common keyboard shortcuts for panel navigation, sidebar toggling, History refresh, and Settings.
- Shows shortcut hints directly in the sidebar so the main navigation shortcuts are easier to discover.
- Updates the user guide and behavior contract for the new grouping and shortcut behavior.
