Install from GitHub Releases.

SD Import ships as a free, signed, and notarized DMG for Apple Silicon Macs running macOS 14 or newer.

Quick install.

Download the DMG, move the app into Applications, then choose your import folders on first launch.

Requirements: Apple Silicon Mac, macOS 14 or newer, and enough free space at the selected photo or video destination.

1

Download

Get SD-Import.dmg from the latest GitHub Release.

2

Copy to Applications

Open the DMG and drag SD Import.app into Applications.

3

Choose folders

Open the app and choose source and destination folders during first-run setup.

Download for Apple Silicon Mac View latest GitHub Release

Gatekeeper and first launch.

If macOS says the app was downloaded from the internet, choose Open. If macOS blocks first launch, open System Settings, Privacy & Security, then choose Open Anyway for SD Import.

Requirements

  • Apple Silicon Mac.
  • macOS 14 or newer.
  • SD card, card reader, camera-mounted card, or a selected source folder.
  • Enough free space at your selected photo or video destination.
