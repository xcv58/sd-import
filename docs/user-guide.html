User Guide

Set destinations once, preview each card before copying, and keep a clear record of what SD Import already imported.

Quick start.

Most users only need to install once, set destinations once, and review the preview each time a card is inserted.

1

Install

Download the signed DMG, open it, and drag SD Import.app into Applications.

2

Set folders

Open Settings, choose photo and video destinations, and keep Theme set to System unless you prefer Light or Dark.

3

Import

Insert a card, review the preview, then keep the card connected until copy progress finishes.

Import from a card.

SD Import can appear automatically when a card mounts, or you can open the app and choose Import From Card....

  1. 1

    Connect a card.

    Insert an SD card or connect a camera card reader.

  2. 2

    Review the scan.

    Check what is new, already imported, unsupported, or conflicting.

  3. 3

    Start the copy.

    Confirm the destinations and leave the source connected until the import completes.

Safety notes

  • SD Import copies files; it does not erase the card.
  • Files already imported are skipped on future runs.
  • If a destination file already exists but is different, SD Import keeps both files by adding a suffix.
  • Leave the app in Applications so updates can install correctly.

Keyboard shortcuts

  • Command-I: open Import.
  • Command-1 through Command-4: Import, History, Settings, Diagnostics.
  • Control-Tab: next panel.
  • Command-R: refresh History.
  • Command-,: Settings.

If import does not start automatically.

Make sure Prompt on card mount is enabled in Settings, open SD Import manually, then choose Import From Card.... For persistent issues, export diagnostics and contact support.
