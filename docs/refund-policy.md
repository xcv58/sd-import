# SD Import Refund Policy

Last updated: 2026-05-10

SD Import is currently free. There are no purchases, subscriptions, paid
licenses, or in-app payments to refund.

## Current Public Release

The public release is distributed at no charge through GitHub Releases. Because
no payment is collected, no refund process applies.

## If Distribution Changes Later

If SD Import ever adds paid distribution, subscriptions, or payment
infrastructure, this policy must be updated before release and the applicable
store or payment processor rules will apply.

## Support

For installation or update problems, email [i@xcv58.com](mailto:i@xcv58.com) or
open a GitHub issue.
